## 项目观

我不把单个仓库当作唯一代表作。不同阶段会做不同类型的项目：有的是产品，有的是脚本，有的是基础设施，有的是用来验证技术路线的工程样本。对我来说，更重要的是代码能否被验证、能否持续运行、能否被后来的人读懂和维护。
